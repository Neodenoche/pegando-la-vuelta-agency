What was built

Six systems.
One ecosystem.

🐾
13-tier loyalty system
Members level up automatically based on activity and PNT holdings. Each tier unlocks exclusive perks, roles, and raffle tickets.
🎰
Ruleta Panther
Surprise USDT raffle activated manually via /ruleta_on. No fixed schedule — keeps the community on edge and rewards active members.
🔗
Referral tracking
Operación 1,000 Cazadores — unique invite links per user, tracked in SQLite. Rewards distributed automatically on milestone hits.
🎮
PNT Defender mini-game
HTML5 Space Invaders-style game embedded in the Telegram Mini App. 13 levels, UFO abductors, TR-3B Astra boss. Leaderboard via /game-leaderboard endpoint.
📊
Stats & leaderboard
Live community rankings, personal stats, and game scores — all queryable by members directly in chat.
⚡
Railway + GitHub CI/CD
Webhook mode, persistent SQLite at /data/panther.db, auto-deploy from GitHub. Zero downtime updates.
